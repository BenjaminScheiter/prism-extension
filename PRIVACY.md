# Privacy Policy

Prism is a local-first Chrome extension for rewriting prompts before they are sent to AI chat apps.

## Data Collection

Prism does not sell user data and does not use prompt content for advertising, profiling, or third-party analytics.

The extension stores only local settings, per-site enabled state, and local prompt metrics using Chrome extension storage.

## Prompt Processing

Prompt rewriting runs locally inside the extension package. Prism V1 does not call external AI APIs from the extension package.

## Site Access

Prism auto-runs only on supported AI chat hosts declared in the extension manifest. Users may explicitly enable Prism on another HTTPS site from the popup. Those additional grants use Chrome optional permissions and can be removed by the user.

## Contact

For questions about this project, use the public GitHub repository issue tracker.
